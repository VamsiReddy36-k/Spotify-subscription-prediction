# 🎧 Spotify User Subscription Prediction & Churn Analysis

An end-to-end Machine Learning project that predicts whether a Spotify user is likely to remain **Active** or become **Inactive** based on listening behavior, engagement patterns, subscription type, and user preferences.
This project also provides churn risk analysis through an interactive Streamlit dashboard.

# 🚀 Project Overview

The goal of this project is to help music streaming platforms improve user retention by identifying users who are at risk of becoming inactive.
Using Machine Learning techniques, the application analyzes user activity and predicts user subscription behavior in real time.

# 📌 Features

✅ Predicts whether a user is **Active** or **Inactive**
✅ Calculates user churn probability
✅ Interactive Spotify-themed Streamlit dashboard
✅ Real-time predictions using trained ML model
✅ User-friendly interface for data input and analysis
✅ Feature engineering and preprocessing pipeline integration
✅ Machine Learning model deployment with Streamlit
